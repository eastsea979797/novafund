# NovaFund

## 프로젝트 소개
NovaFund는 사용자가 프로젝트를 등록하고 펀딩에 참여할 수 있는
참여형 펀딩 플랫폼입니다.
본 프로젝트는 팀 프로젝트로 진행되었으며,
저는 인증/회원/관리자 기능을 담당했습니다.

## 기술 스택

##Backend
- Spring Framework(MVC)
- Spring Security

##Frontend
- JSP
- HTML/CSS/JavaScript

##Database
- Oracle DB

##Server/Tool
-Apache Tomcat
-Git / Github

## 담당 기능
- 로그인 / 회원가입
- 마이페이지
- 관리자 페이지

## 실행 방법
- src/main/resources/db.properties.example 파일을
  db.properties로 복사 후 DB 계정 정보를 입력합니다.
- JDK 8 이상
- Tomcat 서버를 실행합니다.
